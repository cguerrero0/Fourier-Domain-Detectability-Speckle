# Fourier-Domain-Detectability-Speckle

## Summary

This repository presents a reproducible semi-analytical framework to study the detectability of binary stars in speckle interferometry from a Fourier-domain perspective.

The code accompanies the manuscript:

**“A Fourier-Domain Framework for the Detectability of Binary Stars in Speckle Interferometry”**

The repository is designed to illustrate and reproduce the main figures and simulations discussed in the paper, including:

- one-dimensional visibility curves,
- two-dimensional binary power-spectrum maps,
- illustrative binary autocorrelation maps,
- detectability maps in the \((q,\Delta m)\) plane,
- sampling-dependent detectability simulations,
- attenuation-dependent detectability simulations.

The implementation is intentionally semi-analytical. It focuses on the dominant Fourier-domain ingredients that govern practical detectability, rather than on a full end-to-end observational reduction pipeline.

---

## Purpose of the repository

This repository is intended to provide:

- a reproducible implementation of the simulations presented in the manuscript,
- a compact pedagogical tool for understanding binary-star signatures in Fourier space,
- a practical framework for exploring how detectability changes with separation, contrast, attenuation, noise, and detector sampling.

## Workflow (recommended order)

1. Open and run the notebook `binary_speckle_detectability.ipynb`.
2. Execute the blocks sequentially to generate:
   - 1D visibility curves,
   - 2D binary power spectra,
   - illustrative autocorrelation maps,
   - detectability maps with increasing degradation,
   - detectability maps for different sampling regimes,
   - detectability maps for different effective attenuation levels.
3. Save the resulting figures locally or to Google Drive if using Colab.
